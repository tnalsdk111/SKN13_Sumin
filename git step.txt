- git pull  : 원격에 새로 추가된 내역을 내 로컬에 받기.
- 변경 후
   - git status: 로컬 저장소의 상태 조회
   - git add : staging area(index) 로 옮기기. 
        - git add .
   - git  commit   -m   "커밋메세지"
	- git commit -m "두개 파일 수정"
   - git push  서버이름  브랜치
        - git push  origin  main

- 여러분 repository
   1. 파일 생성   1-2. 생성한 파일을 삭제
   2. git status 확인
   3. git add
   4. git commit 
   5. git push
